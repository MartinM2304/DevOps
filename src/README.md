# Управление на изисквания - НЕ финална версия

Съдържание на архива: 

- 46.1_README.txt   - текущият файл
- WEB               - директорията с изходния код

## Инструкции (без XAMPP)
1. Клонирате кода
3. Настройте конфигурацията за достъп до базата. Пример: 
```php
<?php
return [
    'DB_HOST' => 'localhost',
    'DB_NAME' => 'requirements_db',
    'DB_USER' => 'root',
    'DB_PASS' => ''
];
```
2. Създавате базата използвайки `setup_db.php` скрипта (`php setup_db.php`), който ще постави и примерни данни
3. Стартирате сървър с `php -S localhost:8000` (портът не е от значение)
4. Системата е достъпна на http://localhost:8000/frontend/index.html?

## Инструкции XAMPP
1. Сложете папката WEB в /httdocs в XAMPP
2. Настройте базата през интерфейса на XAMPP или чрез конзолата от XAMPP приложението, използвайки скрипта `setup_db.php`
